### 自定义模型下载地址

OCR 模型下载地址支持配置化，方便更换镜像或指向自有仓库，无需重新编译。

**编辑** 应用的 `config.json` 文件（路径见下表），修改 `modelDownloadUrls` 数组。程序按顺序尝试每个地址，下载失败自动切换到下一个。`{model}` 是占位符，会被替换为 `ppocr-v4` / `ppocr-v5` / `ppocr-v6`。

| 系统 | 配置文件路径 |
|------|-------------|
| Windows | `%APPDATA%\LynxOCR\config.json` |
| macOS | `~/Library/Application Support/LynxOCR/config.json` |
| Linux | `~/.local/share/LynxOCR/config.json` |

示例 `config.json`：

```json
{
  "modelPath": "C:\\Users\\你的名字\\AppData\\Roaming\\LynxOCR\\models",
  "activeOcrModel": "ppocr-v6",
  "sidebarCollapsed": false,
  "ocrScreenshotShortcut": "Ctrl+Shift+O",
  "modelDownloadUrls": [
    "https://gitcode.com/tabortao/LynxOCR/releases/download/model/{model}.zip",
    "https://gitee.com/tabortao/LynxOCR/releases/download/model/{model}.zip",
    "https://code2tabor.s3.bitiful.net/PPOCR/{model}.zip"
  ]
}
```

若字段缺失会恢复默认地址。修改后重启应用即可生效。
